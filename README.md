## 앱 소개
### 앱의 방향성, 취지
- 간편하고 시각적인 요소들을 강조한 일정관리 앱
- 간단한 조작으로 일정을 처리해 나가게 함으로써 일정을 수행하는 일이 하나의 업무로 느껴지지 않도록 함
- 일정의 성취 정도를 시각적으로 확인할 수 있으므로 일정 수행에 흥미를 느낄 수 있도록 장려, 스케줄을 효과적으로 구성하도록 독려
